Citi Files

Citi sends us three data files every month with net of fees performance, gross of fees performance, and benchmark monthly returns. These files need to be uploaded into system.

Web User Interface

The users would like a web interface to check the performance numbers after they are uploaded.

Every month we need to send a data feed of net performance to the company Omni. They help distribution mutual fund performance.

Excel Performance Report

Users would like an formatted Excel report that shows fund performance run on demand.
