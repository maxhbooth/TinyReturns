Add Readme File

The development team would like us to add a "read me" file that documents how to get started with this project.

  • File should be called 'readme.md' in the root folder.
  • It should include as much information as possible for a new developer to modify Tiny Returns.
  • The file content should format the text with markdown. Click here for markdown help.
