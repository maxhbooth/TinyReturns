Web Report Improvements

As a user I would like the following enhancements made to the web report...

  • We would like the ability to report on past months.
  • We would like all values in percentage format rounded to two decimal places. Default to last complete month.
  • We would like the ability to choose net or gross return types. Default to net.

Benchmark Disclosures

As a user I must (by law) include the disclosures for all benchmarks on all reports going to the public.

Standard Deviation

As a FAS user I would to include a standard deviation value with the global adviser reports.

  • Use the population calculation type of standard deviation => Standard Deviation
  • You must include an average return value (mean) for the standard deviation value to have meaning.
  • All FAS teams use both the Web Report and the Excel report to create their adviser reports.
  • You must have at least 12 months of returns for a standard deviation value to be valid.
