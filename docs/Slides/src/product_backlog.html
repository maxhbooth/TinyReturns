Add Since First Full Month Return

As an Institutional team user I would like to include a Since First Full Month return value for the portfolio and related benchmarks in our reports.

  • All Institutional teams use both the Web Report and the Excel report to create reports.
  • Do not assume that the list of monthly return values attached to the portfolio is the exact number of months that is needed to calculate the return. Instead use in the inception date of the portfolio to find the EXPECTED first full month return.
  • The related benchmarks should use the same time period portfolio since first full month.

Standard Deviation

As a FAS user I would to include a standard deviation value with the global adviser reports.

  • Use the population calculation type of standard deviation => Standard Deviation
  • You must include an average return value (mean) for the standard deviation value to have meaning.
  • All FAS teams use both the Web Report and the Excel report to create their adviser reports.
  • You must have at least 12 months of returns for a standard deviation value to be valid.
